## 🧠 About The Project

MindSpace is a revolutionary **free, open-source mental health platform** that combines:
- **3D brain visualization** (Three.js)
- **Voice-guided therapy sessions** (Web Speech API)
- **15+ evidence-based techniques** (breathing, mindfulness, CBT, grounding)
- **Emergency crisis protocols** (instant grounding)

Three integrated platforms in one:
1. **Neuromatrix** - Cognitive mapping & brain visualization
2. **NeuroZen** - 3D therapy & mental wellness
3. **ZenScape** - Abstract art & stress relief

**Why?** Because mental health care should be free, accessible, and private for everyone.

**1 in 8 people** suffer from mental disorders. **75% receive NO treatment**. We're changing that.

## ✨ Key Features

### 🧠 Neuromatrix - Cognitive Mapping
- Real-time 3D brain visualization (Three.js)
- Cognitive pattern analysis
- Neural network simulations
- Personality & intelligence mapping

### 🧘 NeuroZen - 3D Therapy
- Voice-guided breathing techniques (4-7-8, Box, Coherent)
- CBT tools & thought challenging
- Emergency panic protocols
- Progress tracking & analytics

### 🌿 ZenScape - Stress Relief
- Abstract art therapy gallery
- Therapeutic soundscapes
- Guided meditation
- Ambient music player

### 🎯 Platform Features
- ✅ 100% Free - No subscriptions, no paywalls
- ✅ Voice control (Web Speech API)
- ✅ Works offline (PWA ready)
- ✅ Mobile responsive
- ✅ Privacy first - All data stored locally
- ✅ Accessibility features (keyboard, screen reader)
